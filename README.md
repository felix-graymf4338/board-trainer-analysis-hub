## What is Board Trainer?

Board Trainer is aimed at poker study sessions that revolve around board texture and GTO practice. Since it runs in the browser, you can open it directly in HTML form and get started without installing a heavy desktop application.

This project keeps things intentionally focused: it gives players a simple way to revisit boards, repeat training drills, and stay organized during structured practice. It is a practical fit for anyone building a repeatable routine around poker decision-making and board review.
